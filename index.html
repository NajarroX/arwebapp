Instrucciones de Uso

  1. Selecciona el tipo de modelo (estático o animado) y el formato (GLB/GLTF o FBX)
  2. Elige un modelo de la lista o sube tu propio modelo 3D
  3. Ajusta la escala según tus necesidades
  4. Selecciona el tipo de tracking (Target o World)
  5. Haz clic en "Cargar Modelo" para previsualizarlo
  6. Haz clic en "Iniciar Experiencia AR" para activar la cámara
  7. Para Target Tracking: Enfoca la cámara a un marcador o imagen de referencia
  8. Para World Tracking: Explora tu entorno para colocar el modelo
  9. Usa los botones de grabación para capturar tu experiencia AR como MP4
